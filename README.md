# MW Tecnologia Estratégica — Site institucional

Site institucional **moderno, minimalista e profissional** para a **MW Tecnologia Estratégica**, com foco em gerar autoridade, transmitir confiança e converter visitantes em pedidos de **diagnóstico gratuito**.

## Posicionamento

Organização e estabilidade tecnológica para pequenos negócios que não podem parar.

**Público-alvo:** clínicas, salões de beleza, estúdios de estética, pequenos comércios e profissionais liberais.

## Estrutura da página

1. **Hero** — Título e subtítulo do posicionamento + botões (Solicitar diagnóstico gratuito / Conheça nossos serviços)
2. **Problemas** — “Sua empresa enfrenta algum desses problemas?” + lista
3. **Solução** — “Como a MW resolve isso” + itens com ícones
4. **Método** — “Nosso método” (4 passos)
5. **Para quem é** — Público-alvo em destaque
6. **Sobre a empresa** — Texto institucional
7. **Diferencial** — Frase de posicionamento
8. **CTA final** — “Vamos estruturar sua tecnologia?” + botão
9. **Contato** — Formulário, WhatsApp, e-mail
10. **Rodapé** — Contato, WhatsApp, redes sociais

Extras: **WhatsApp fixo** no canto inferior direito.

## Design

- **Paleta:** azul profundo (#0B3D5C), cinza grafite (#2E2E2E), branco predominante, verde (#2E8B57) para destaques e botões
- **Tipografia:** Inter (moderna e limpa)
- **Estilo:** minimalista, corporativo, clean, com bastante espaço em branco
- **Animações:** entradas suaves (fadeUp) nas seções

## Como rodar

1. Abra a pasta do projeto e sirva com um servidor local (ex.: extensão “Live Server” ou `npx serve .`).
2. Acesse o endereço indicado no navegador.

## Antes de publicar

1. **Contato:** no `index.html`, substitua:
   - `contato@mwtecnologia.com.br` pelo seu e-mail
   - `https://wa.me/5500000000000` pelo seu número (55 + DDD + número, sem espaços)
2. **Redes sociais:** atualize os links do rodapé (Instagram, LinkedIn) ou remova os que não usar.
3. **Formulário:** integre o envio a um backend ou serviço (Formspree, API etc.).

## Requisitos atendidos (conforme prompt)

- Site responsivo (mobile first)
- Design elegante e profissional
- Animações suaves e transições
- Ícones minimalistas
- Código organizado
- SEO básico (meta description, título, estrutura de headings)
